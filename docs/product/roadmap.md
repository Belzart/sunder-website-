# Roadmap

## Current State: v0.3 — Lab Foundation

The site has a working home experience, research page, access page, field interaction system, ambient lab atmosphere, and hidden discovery layers. The core thesis is expressed through interaction, not just copy.

## v0.4 — Material Connection (Next)

Goal: Make the blob influence the rest of the interface directly. Break the separation between the 3D specimen and the HTML content.

- [ ] Application preset switching — hovering a domain card changes the blob's material
- [ ] Blob position projected to screen space, fed into FieldProvider
- [ ] Text near the blob center warps subtly
- [ ] Page transitions — content fades, blob morphs, new content appears
- [ ] Keyboard shortcuts for material presets (1-4 keys)

## v0.5 — Experimentation Layer

Goal: Make the site feel playable. Users can run small experiments on the specimen.

- [ ] Pressure test — click-and-hold on blob compresses it, shows pressure readout
- [ ] Temperature slider — shifts material between cold/room/hot presets
- [ ] Resonance discovery — rapid clicks trigger high-frequency wobble
- [ ] Extended hold-to-reveal on research cards
- [ ] Mini lab control panel (hidden, keyboard-toggled)

## v0.6 — Depth and Discovery

Goal: Make the experience feel nearly infinite through hidden layers.

- [ ] Slow data stream — terminal log scrolling at viewport edge
- [ ] Secret /lab route with experimental interactions
- [ ] Cursor stillness reactions (blob reaches toward still cursor)
- [ ] More manifesto fragments and micro-copy throughout
- [ ] Scroll-velocity responsive ambient motion

## v0.7 — Polish and Performance

Goal: Production-ready quality across all devices.

- [ ] Mobile/touch optimization (gesture-based blob interaction)
- [ ] Performance mode for low-power devices
- [ ] Sound design (opt-in, very subtle)
- [ ] OG image generation from blob screenshot
- [ ] SEO + structured data
- [ ] Analytics events (blob interaction, page depth, access clicks)

## v1.0 — Launch

Goal: Confident stealth launch. Send to investors, partners, talent.

- [ ] All core pages complete and polished
- [ ] 10+ hidden interactions implemented
- [ ] Custom domain configured
- [ ] Vercel production deployment
- [ ] Email capture or access request form
- [ ] Final copy pass with external review
- [ ] Cross-browser and device testing

## Decisions Deferred

- Whether to add an /applications page (may combine with /research)
- Whether to add team/about content (likely not for v1.0)
- Whether to add a blog or "Lab Notes" section
- Sound design scope (high risk of being annoying)
- Whether the /brand page should be password-protected
